## 기술 스택

| 항목 | 내용 |
|------|------|
| 백엔드 | Node.js + Express |
| 프론트엔드 | 바닐라 HTML/CSS/JS |
| Excel 파싱 | xlsx |
| Excel 생성 | exceljs |
| 메일 발송 | nodemailer (네이버 SMTP, port 465) |
| 홈택스 자동화 | puppeteer-core + 시스템 Chrome |
| 데이터 저장 | JSON 파일 (data/ 폴더) |

### 네이버 SMTP
- Host: smtp.naver.com / Port: 465 / secure: true
- 비밀번호: 네이버 로그인 비밀번호
- 사전 설정: 네이버 메일 → 환경설정 → POP3/SMTP → 사용함

### 홈택스 봇
- puppeteer-core + 시스템 Chrome 사용
- 로그인은 사용자가 직접 (공동인증서)
- 로그인 감지 후 자동으로 폼 입력 및 최종 발행
- UI 변경 시 hometaxBot.js 셀렉터 수정 필요할 수 있음
