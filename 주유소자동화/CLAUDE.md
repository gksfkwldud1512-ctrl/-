## 웹앱 기능 설명

### 탭 1: 거래명세서 발행
1. 연/월 선택 → 발행일자 자동 설정 (해당 월 다음달 1일)
2. Excel 파일 업로드 (BOS 시스템에서 뽑은 월별 상세거래내역서)
3. 월별 현황 칩에서 어느 달 파일이 있는지 확인 가능
4. 업체 목록: 외상/카드현금/합계 금액 표시, 외상 업체만 체크박스
5. 선택 업체 거래명세서 생성 → 다운로드

### 탭 2: 고객 관리
- 업체명, 사업자번호, 이메일, 연락처 등록/수정/삭제
- 거래명세서 공급받는자 정보 및 메일 발송에 활용

### 탭 3: 메일 발송
- 외상 업체만 표시
- 거래명세서 생성 + 이메일 등록된 업체만 발송 가능
- 선택 후 일괄 발송

### 탭 4: 홈택스 세금계산서
- 외상 업체만 표시
- 사업자번호 등록된 업체만 선택 가능
- 선택 시 Chrome 브라우저 자동 열림 → 공동인증서 로그인 → 폼 자동 입력 → 최종 발행 자동

### 탭 5: 설정
- 네이버 SMTP 이메일/비밀번호 저장 (암호화 없이 settings.json에 저장됨)

### 네이버 SMTP
- Host: smtp.naver.com / Port: 465 / secure: true
- 비밀번호: 네이버 로그인 비밀번호 (앱 비밀번호 아님)
- 사전 설정: 네이버 메일 → 환경설정 → POP3/SMTP → SMTP 사용함

### 홈택스 봇 주의사항
- puppeteer-core 사용 (시스템 Chrome 필요)
- Chrome 경로 자동 탐색: Program Files/Google/Chrome/Application/chrome.exe
- 로그인은 사용자가 직접 (공동인증서)
- 로그인 감지 후 자동으로 세금계산서 폼 입력 및 최종 발행
- 홈택스 UI 변경 시 hometaxBot.js 셀렉터 수정 필요할 수 있음
