📊 탱크 실재고량 파일 양식 안내

BOS ATG(자동탱크측정) 시스템에서 출력한 탱크 실재고량.xlsx 파일을 그대로 업로드합니다.

📌 파일 컬럼 구조 (자동 인식)
  • M열 (13번째) — 휘발유 합계(L): 모든 휘발유 탱크 ATG 합산
  • N열 (14번째) — 경유 합계(L): 모든 경유 탱크 ATG 합산
  • O열 (15번째) — 등유 합계(L): 등유 탱크 ATG 값
  • A열 (1번째) — 날짜 (YYYY-MM-DD 형식 또는 Excel 날짜)
📐 재고 오차 계산 방법
재고차(D일) = 전일ATG(D-1) − 당일ATG(D) + D일 입고량
            ※ 입고량은 새벽 배송으로 당일 ATG에 이미 포함 → 빼서 순소비 산출
오차(D일) = 재고차 − BOS판매량(D일)
손익금액 = 오차(L) × FIFO 매입단가
양수(+) = 탱크>BOS (온도팽창·증발손실) / 음수(-) = BOS>탱크 (온도수축·ATG오차·미기록입고)

※ 영업이익에 반영되지 않는 참고 지표입니다. 예) 6/20: 전일ATG 74,327 − 당일ATG 68,716 = 재고차 5,611L / BOS 5,704L / 오차 −93L

💰 영업이익 계산 방식

영업이익 = (휘발유 매출 − 휘발유 판매량 × FIFO 매입단가)
          + (경유 매출 − 경유 판매량 × FIFO 매입단가)
          + (등유 매출 − 등유 판매량 × FIFO 매입단가)
          + 세차 매출
          + 유외상품 매출 − 유외상품 원가
          − 카드 수수료
FIFO 매입단가 결정 순서
  1. 마감자료 판매관리 시트 일별 단가 (우선)
  2. 입고이력(정산단가) 기반 FIFO 계산 (fallback)
정산단가 기간 구분
~3/12휘발유 1,754 / 경유 1,742
3/13~3/26휘발유 1,724 / 경유 1,713
3/27~휘발유 1,934 / 경유 1,923
